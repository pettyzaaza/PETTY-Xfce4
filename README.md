# Back up configuration

A complete, portable XFCE desktop environment configuration backup...

## Features
- **Theme**: Nashville96 (Kanagawa, Solarized, and Steam variants)
- **Terminal**: Kitty (Set as default)
- **Utility**: Fastfetch (Custom configuration included)
- **Icons**: SE98 Icon Pack
- **Fonts**: BigBlueTerm437 Nerd Font (System) & Sarabun (Thai)
- **Whiskermenu**: Pre-configured with `Super` key binding.
- **Thai Support**: Global Sarabun font rendering for browsers and system.
- **Layout**: Dual US/Thai keyboard layout (`Alt+Shift` toggle).

## Installation

1. Clone the repository.
2. Run the installer:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

## Portability
This configuration is designed to be portable. The installer automatically detects your username and updates all Xfconf XML files and shell paths accordingly using the `@USERNAME@` placeholder.
